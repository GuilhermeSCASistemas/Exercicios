# Exercicios

Repositório criado para organizar minhas entregas dos diversos cursos que realizo.

## Organização

Todas as entregas devem ficar **apenas neste repositório**, seguindo a estrutura:

```text
entregas/
  nome-do-curso/
    modulo-ou-aula/
      atividade/
```

Crie uma pasta para cada curso dentro de `entregas/` e mantenha os arquivos de cada entrega dentro da respectiva pasta.
